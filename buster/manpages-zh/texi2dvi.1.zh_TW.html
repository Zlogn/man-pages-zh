TEXI2DVI(1) User Commands TEXI2DVI(1)

NAME¶

texi2dvi - 打印 Texinfo 文檔

SYNOPSIS 總覽¶

texi2dvi [OPTION]... FILE...

DESCRIPTION 描述¶

依次從 Tex 系統中運行每個 Texinfo 或者 LaTex 文件 FILE，直到解決了所有的交叉引用，建立所有的索引。每個包含 FILE 的目錄都將被搜索，查找被包含的文件。FILE 的後綴被用來判斷它的語言 (LaTex 還是 Texinfo)。

需要的時候，在運行 Tex 系統之前，使用 makeinfo 來進行 Texinfo 宏擴展。

操作模式：¶

-b, --batch
非交互式
-c, --clean
刪除所有的輔助文件 (auxiliary)
-D, --debug
打開 shell 調試 (設置 -x)
-h, --help
顯示此幫助，然後退出
-o, --output=OFILE
將輸出保存於 OFILE (隱含着 --clean)；這種情況下只能指定一個輸入文件 FILE
-q, --quiet
不顯示輸出，除非發生錯誤 (隱含着 --batch)
-s, --silent
與 --quiet 相同
-v, --version
顯示版本信息，然後退出
-V, --verbose
報告每一步行爲

調整 TeX：¶

-@
使用 @input 而不是 \input；只對應預先加載的 Texinfo
-e, -E, --expand
強制使用 makeinfo 進行宏擴展
-I DIR
搜索 DIR，查找 Texinfo 文件
-l, --language=LANG
指定文件 FILE 的語言 LANG (值爲 LaTex 或 Texinfo)
-p, --pdf
使用 pdftex 或者 pdflatex 來處理
-t, --command=CMD
在輸入文件的拷貝中插入 CMD；
或 --texinfo=CMD
累積多個值 (?)

環境變量 BIBTEX, LATEX (或 PDFLATEX), MAKEINDEX, MAKEINFO, TEX (或 PDFTEX), 以及 TEXINDEX 的值將用來運行對應的命令，如果設置了它們的話。任何 CMD 字串都被添加到 Texinfo 輸入文件的 @setfilename 之後，或者 LaTex 輸入文件的第一行。

REPORTING BUGS 報告錯誤¶

將錯誤報告發送到 bug-texinfo@gnu.org，一般的問題和討論則發送到 help-texinfo@gnu.org。 Texinfo 主頁：../../software/texinfo/

SEE ALSO 參見¶

texindex 的全部文檔以 Texinfo 手冊頁形式保存。如果你的主機上正確安裝了 info 和 texindex 程序，命令
info texindex

將使你可以讀取完整的手冊。
